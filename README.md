# 컴퓨터네트워크 팀프로젝트 (SMTP)

### 실행

```
javac -d . SendEmail/*.java -encoding utf-8
java SendEmail/SMTP
```

<br/>

### 사용 방법

- 발신자, 수신자의 id에는 각각의 이메일 주소를 기입한다.
- 발신자 Server에는 SMTP 서버 주소를 기입한다.
  - gmail: smtp.gmail.com
  - naver: smtp.naver.com
- 파일을 첨부할 시 `open file` 버튼을 눌러 첨부하고자 하는 파일을 선택한다.
